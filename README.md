# Weather-App

A simple and responsive weather application built using HTML, CSS, and JavaScript.  
It fetches real-time weather data using the OpenWeather API.

## Features

- Search weather by city name
- Displays temperature, condition, humidity, and wind speed
- Responsive user interface
- Real-time API integration

## Tech Stack

- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Run

1. Add your OpenWeather API key in `script.js`.
2. Open `view.html` using Live Server.
3. Enter a city name and click Search.
